# 實用工具與小遊戲網站

一個集合了各種實用工具和有趣小遊戲的網站，完全在瀏覽器中運行，無需下載任何軟件。

## 功能特性

### 實用工具

- **計算機** - 基礎的四則運算計算工具
- **單位轉換** - 支援長度、重量、溫度的單位轉換
- **顏色轉換** - HEX、RGB、HSL 顏色格式轉換
- **JSON 格式化** - JSON 驗證、格式化和壓縮工具
- **密碼生成器** - 安全的隨機密碼生成工具
- **二維碼生成** - 快速生成 QR Code

### 小遊戲

- **2048** - 經典的數字合併遊戲
- **記憶翻牌** - 考驗記憶力的配對遊戲
- **井字棋** - 與電腦對戰的經典遊戲
- **猜數字** - 猜測隱藏的數字遊戲
- **貪食蛇** - 經典的貪食蛇遊戲
- **飛翔的鳥** - 躲避障礙物的遊戲

## 特點

- ✨ **完全響應式設計** - 支援手機、平板和電腦等各種設備
- 🚀 **快速加載** - 純前端應用，無需後端服務
- 🎨 **現代化界面** - 美觀的設計和流暢的動畫
- 📱 **移動友好** - 優化的移動設備體驗
- 🔒 **隱私保護** - 所有數據在本地處理，不上傳任何信息

## 使用方法

1. 在瀏覽器中打開 `index.html`
2. 選擇要使用的工具或遊戲
3. 按照提示進行操作

## 技術棧

- HTML5
- CSS3（響應式設計）
- Vanilla JavaScript（無框架依賴）

## 部署

### GitHub Pages 部署

1. 創建一個 GitHub 倉庫
2. 將所有文件推送到倉庫
3. 在倉庫設置中啟用 GitHub Pages
4. 選擇 `main` 分支作為發佈源
5. 您的網站將在 `https://yourusername.github.io/repository-name` 上可用

### 本地運行

只需在瀏覽器中打開 `index.html` 文件即可。
